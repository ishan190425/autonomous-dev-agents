## Overview

The ADA Agent Dashboard provides 24/7 observability into autonomous development teams. It visualizes role activity, memory state, and cross-repo coordination in real-time.

### Design Principles

1. **Glanceable** â€” Status at a glance, details on drill-down
2. **Non-Intrusive** â€” Monitor without disrupting the autonomous loop
3. **Multi-Repo** â€” Switch context between projects seamlessly
4. **Memory-Aware** â€” Visualize the shared brain, not just events
5. **Character-Driven** â€” Each role has personality (avatars, colors)

## Implementation Notes

### Tech Stack

- **Framework:** Next.js 14+ (App Router)
- **Styling:** Tailwind CSS + shadcn/ui components
- **State:** Zustand for client state, SWR for data fetching
- **Real-time:** File watchers via Node.js + SSE to browser
- **Markdown:** react-markdown with GFM support

### Phase 1 MVP (Sprint 2)

1. Overview page with static file reads
2. Agent cards with role colors
3. Memory bank viewer (markdown render)
4. Single repo support

### Phase 2 (Sprint 3)

1. Real-time file watching + SSE
2. Activity feed with filters
3. Role detail pages
4. Multi-repo switcher

### Phase 3 (Post-Launch)

1. Heat map integration (#118)
2. GitHub webhook integration
3. Historical metrics charts
4. Custom dashboards
