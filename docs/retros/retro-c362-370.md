## Lessons Identified

### L132: Retro cadence requires structural gate, not just documentation

The 5-cycle target was documented in the Scrum playbook as a FIRST CHECK, but actual cadence drifted to 9-10 cycles. Enforcement requires the gate to be mandatory in DISPATCH.md, not optional in playbook.

### L133: Sprint prep docs should reference each other explicitly

When 4+ roles produce related docs in consecutive cycles, each doc should reference prior docs to build a coherent narrative. C370 did this well (referenced C343, C363, C369); other docs could improve cross-linking.

### L134: ASCII wireframes in markdown are surprisingly effective

C365 demonstrated that ASCII diagrams communicate layout to developers without design tool overhead. Version-controlled, diff-able, reviewable in PRs. (Reinforces L127)
